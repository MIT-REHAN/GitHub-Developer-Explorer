GitHub Token Configuration

To bypass the unauthenticated rate limit of 60 requests/hour, you can add a GitHub Personal Access Token (PAT). This increases your rate limit to 5,000 requests/hour.

No permissions (scopes) are required for your token. It is stored locally in your browser and is only sent directly to GitHub's secure API.
